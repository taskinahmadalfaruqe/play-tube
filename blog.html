Q: Discuss the scope of var, let, and const.

Answer:

var:
JavaScript Variables can be declared in 4 ways. var is one of them. but is have some rejection. var is not a block-scoped. we can get value any where is our site. It Hoisting the value

let:
The keyword let in Javascript is used to declare a block-scoped variable in JavaScript. when we can declare a variable with let we can't access it outside the function or statement. But our variable is global we can access it anywhere in js. It can't Hoisting the value.

const:
The keyword const in Javascript is used to declare a block-scoped variable in JavaScript. Change the elements of constant array, Change the properties of constant object. but cant do Reassign a constant value, Reassign a constant array, Reassign a constant object and its also inaccessible outside the statement or function. It can't Hoisting the value.
